# Suku AI

Suku AI is an intelligent conversational client-side assistant built using vanilla web technologies (HTML, CSS, JavaScript). It utilizes an NLP engine and a massive client-side knowledge base that adapts and grows as you chat or train it.

## Features
- Client-side Natural Language Processing (NLP)
- Dynamic learning and teaching commands (e.g., "learn that [question] means [answer]")
- Math expression evaluation
- Context-aware dialogue memory
- Interactive training interfaces and real-time learning logs
- Clean responsive layout using custom CSS variables, dark mode styling, and fluid motion design
